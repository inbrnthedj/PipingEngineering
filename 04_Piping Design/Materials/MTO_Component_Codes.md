# MTO Component Code Cheat Sheet

When taking off an MTO, materials change their standard ASTM name prefix depending on the **product component form** (how the piece is manufactured). Use this cheat-sheet to instantly identify components.

## Material Grouping Reference Table

| Material Type | Pipe Spec | Flange / Forged Valve Spec | Fitting Spec (Elbows/Tees) | Cast Valve Spec ($\ge$ 2") |
| :--- | :--- | :--- | :--- | :--- |
| **Carbon Steel (CS)** | **A106** / A53 | **A105** | **A234 WPB** | **A216 WCB** |
| **Stainless Steel (SS)** | **A312** (TP316) | **A182** (F316) | **A403** (WP316) | **A351** (CF8M) |
| **Alloy Steel (Chrome)** | **A335** (**P11**) | **A182** (**F11**) | **A234** (**WP11**) | **A217** (**WC6**) |
| **Low-Temp Carbon (LTCS)**| **A333** Gr. 6 | **A350** LF2 | **A420** WPL6 | **A352** LCC/LCB |

## Summary Code Decoding Trick

* **P-Codes (P11, P22):** Always indicate an **Alloy Pipe** specification.
* **F-Prefix (F316, F11) or A105:** Always stands for a **Forging**. This identifies **Flanges** or small socket-weld valves.
* **WP-Prefix (WPB, WP316, WP11):** Always stands for **Wrought Piping**. This identifies an **Elbow, Tee, or Reducer (Fitting)**.
* **W-C-Prefix (WCB, WC6) or CF8M:** Always stands for a **Casting**. This identifies large, cast **Valve Bodies**.
