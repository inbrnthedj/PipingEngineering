# ASME B31.3 Component Standards Reference

This document provides a structured reference table for piping components manufactured in accordance with listed standards as mandated by **ASME B31.3 (Chapter IV, Paragraph 326)**. 

Components manufactured to these standards are inherently acceptable for use within their stated pressure-temperature ratings under Normal Fluid Service requirements.

## Technical Application Rules
1. [cite_start]**Listed vs. Unlisted Components:** Components conforming strictly to this table are designated as **Listed Components**[cite: 1403, 1404]. [cite_start]Any design variation or component outside these tables must follow the rigorous evaluation requirements for **Unlisted Components** outlined in Paragraph **302.2.3** [cite: 1338] [cite_start]or **304.7.2**[cite: 1345, 1409].
2. [cite_start]**Edition Control:** Specific dates/editions for these standards are intentionally omitted in the code body text to avoid obsolescence and are officially tracked under **Appendix E (Reference Standards)**[cite: 1413, 1414, 1424, 1425].
