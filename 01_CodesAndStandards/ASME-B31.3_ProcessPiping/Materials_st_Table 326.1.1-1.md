# ASME B31.3 Metallic Component Standards Reference
> **Standard Reference:** Table 326.1.1-1 (Formerly Table 326.1)  
> **Scope:** Chapter IV — Standards for Piping Components

This document provides a structured reference table for metallic piping components manufactured in accordance with listed standards as mandated by **ASME B31.3 Chapter IV**. Components manufactured strictly to these standards are inherently acceptable for use within their stated pressure-temperature ratings under Normal Fluid Service requirements without requiring extra design validation.
